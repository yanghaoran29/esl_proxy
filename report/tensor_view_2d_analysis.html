🎯 2. tensor_view_2d 主函数详解

注意：根据代码分析，tensor_view_2d 函数并未在 tensor_ops.py 中直接定义。 相反，该模块提供了多个构建二维张量视图的核心函数。

💡 核心发现

在 PyPTO 框架中，构建 2D 张量视图通常通过以下组合实现：

  • • create() - 创建指定形状的张量
  • • slice() - 从现有张量提取子视图
  • • reshape() - 改变张量形状
  • • transpose() - 交换维度

核心函数 1: create() - 张量创建

def create()

创建一个新的张量，指定形状和数据类型

def create( shape: Sequence[int | Expr] | _ir_core.MakeTuple, dtype: DataType, layout: TensorLayout = TensorLayout.ND, manual_dep: bool = False, span: Span | None = None, ) -> Call:

参数说明

参数名 类型 说明 示例
shape Sequence[int | Expr] | MakeTuple 张量各维度大小 [1024, 512], [H, W]
dtype DataType 元素数据类型 DataType.FP32, DataType.INT32
layout TensorLayout 张量内存布局 (默认ND) TensorLayout.ND, TensorLayout.DN
manual_dep bool 是否手动管理依赖追踪 False (默认)
span Span | None 源码位置（调试用） None (自动捕获)

实现逻辑

def create(shape, dtype, layout=TensorLayout.ND, manual_dep=False, span=None): # 1. 获取或捕获源码位置信息 actual_span = _get_span_or_capture(span) # 2. 将shape转换为MakeTuple格式 shape_tuple = _to_make_tuple(shape, actual_span) # 3. 构建参数和关键字参数 args = [shape_tuple] kwargs = {"dtype": dtype, "layout": layout} if manual_dep: kwargs["manual_dep"] = True # 4. 创建IR调用表达式 return _ir_core.create_op_call("tensor.create", args, kwargs, actual_span)

使用示例

# 创建 2D 张量 tensor_2d = create([1024, 512], DataType.FP32) # 输出: Call expression with TensorType[1024, 512] FP32 # 创建 2D 张量，使用不同的布局 tensor_dn = create([1024, 512], DataType.FP32, layout=TensorLayout.DN)

核心函数 2: slice() - 张量切片

def slice()

从输入张量创建具有新形状和偏移量的切片视图

def slice( tensor: Expr, shape: list[int | Expr] | _ir_core.MakeTuple, offset: list[int | Expr] | _ir_core.MakeTuple, valid_shape: list[int | Expr] | _ir_core.MakeTuple | None = None, drop_dims: Sequence[int | Expr] | None = None, pad_value: PadValue | int | float | None = None, span: Span | None = None, ) -> Call:

关键特性

  • ✅ 零拷贝视图 - 不复制数据，创建一个新的视图
  • ✅ 维度丢弃 - 支持 numpy 风格的维度规约
  • ✅ 填充值支持 - 可选边界填充
  • ✅ 有效形状 - 支持部分有效区域

参数详解

参数 类型 说明
tensor Expr 输入张量表达式
shape list | MakeTuple 新形状维度（全rank，即使标量索引贡献单位维度）
offset list | MakeTuple 切片在原张量中的偏移量
valid_shape list | MakeTuple | None 有效形状（可选，默认空）
drop_dims Sequence | None 要从结果类型中擦除的轴（numpy风格维度规约）
pad_value PadValue | int | float | None 超出有效形状元素的填充模式

实现细节

def slice(tensor, shape, offset, valid_shape=None, drop_dims=None, pad_value=None, span=None): actual_span = _get_span_or_capture(span) # 转换参数为MakeTuple格式 shape_tuple = _to_make_tuple(shape, actual_span) offset_tuple = _to_make_tuple(offset, actual_span) # 构建操作数列表 args = [tensor, shape_tuple, offset_tuple] # 处理drop_dims（第5个操作数） if drop_dims: # valid_shape（第4个）必须存在，空MakeTuple表示"无valid_shape" args.append(_to_make_tuple( valid_shape if valid_shape is not None else [], actual_span )) args.append(_to_make_tuple(list(drop_dims), actual_span)) elif valid_shape is not None: args.append(_to_make_tuple(valid_shape, actual_span)) # 处理填充值 kwargs = {} if pad_value is not None: kwargs["pad_value"] = (pad_value if pad_value is PadValue.null else normalize_pad_value(pad_value)) return _ir_core.create_op_call("tensor.slice", args, kwargs, actual_span)

核心函数 3: reshape() - 形状变换

def reshape()

改变张量的形状，不改变数据布局

def reshape( tensor: Expr, shape: list[int | Expr] | _ir_core.MakeTuple, valid_shape: list[int | Expr] | _ir_core.MakeTuple | None = None, span: Span | None = None, ) -> Call:

使用示例

# 将 [B, H*W] 张量 reshape 为 [B, H, W] tensor_4d = reshape(tensor_2d, [B, H, W]) # 使用表达式作为维度 batch = dim(tensor_2d, 0) # 获取第一维大小 rest = dim(tensor_2d, 1) # 获取第二维大小 reshaped = reshape(tensor_2d, [batch, rest])

核心函数 4: transpose() - 维度交换

def transpose()

通过交换两个轴来转置张量

def transpose( tensor: Expr, axis1: int | ConstInt, axis2: int | ConstInt, valid_shape: list[int | Expr] | _ir_core.MakeTuple | None = None, span: Span | None = None, ) -> Call:

特性说明

支持负索引：axis1 和 axis2 参数都支持负数索引， 例如 axis1=-1 表示最后一个维度，axis2=-2 表示倒数第二个维度。

示例

# 转置 2D 张量的两个维度 [M, N] -> [N, M] transposed = transpose(tensor_2d, 0, 1) # 等效的负索引写法 transposed = transpose(tensor_2d, -2, -1)

3.3 _normalize_expr()

_normalize_expr()

标准化表达式，将原始数据类型转换为适当的 Expr 类型

def _normalize_expr(value, actual_span, int_dtype=DataType.INDEX, float_dtype=DataType.FP32): """ 将Python原生类型转换为IR表达式 转换规则: - int -> ConstInt (指定数据类型) - float -> ConstFloat (指定数据类型) - Expr -> 直接返回 - 其他 -> 保持原样 """ if isinstance(value, int): return ConstInt(value, int_dtype, actual_span) elif isinstance(value, float): return ConstFloat(value, float_dtype, actual_span) elif isinstance(value, Expr): return value else: return value

3.4 normalize_pad_value()

normalize_pad_value()

标准化填充值，从多种表示形式转换为 PadValue 枚举

# 位置: pypto/ir/op/_pad_value.py def normalize_pad_value(pad_value): """ 填充值标准化 支持的输入: - PadValue枚举: zero, max, min - 数值: 0, inf, -inf - 其他 -> 抛出异常 """ if isinstance(pad_value, PadValue): return pad_value if pad_value == 0: return PadValue.zero elif pad_value == math.inf: return PadValue.max elif pad_value == -math.inf: return PadValue.min else: raise ValueError(f"Unsupported pad_value: {pad_value}")

PadValue 枚举说明

枚举值 数值表示 说明
PadValue.zero 0 用零填充
PadValue.max inf 用正无穷填充
PadValue.min -inf 用负无穷填充
PadValue.null - 不填充（特殊标记）

3.5 resolve_cast_mode()

resolve_cast_mode()

解析类型转换模式，从字符串或整数转换为标准模式值

def resolve_cast_mode(mode): """ 转换模式解析 支持的模式: - 字符串: "none", "rint", "round", "floor", "ceil", "trunc", "odd" - 整数: 0-6 - 其他 -> 抛出异常 """ if isinstance(mode, str): mode_map = { "none": 0, "rint": 1, "round": 2, "floor": 3, "ceil": 4, "trunc": 5, "odd": 6, } return mode_map.get(mode) elif isinstance(mode, int): return mode else: raise TypeError(f"Invalid mode type: {type(mode)}")

5.1 创建 2D 张量的标准模式

# 模式1: 直接创建 2D 张量 tensor_2d = create([rows, cols], DataType.FP32) # 模式2: 从 4D 张量切片获取 2D 视图 # 假设有 tensor_4d [B, H, W, C]，提取第一个样本的特征图 view_2d = slice( tensor_4d, shape=[H, W * C], # 展平剩余维度 offset=[batch_idx, 0, 0, 0], drop_dims=[0] # 丢弃batch维度 ) # 模式3: Reshape 变换 # 将 [B, H, W] 转为 [B*H, W] tensor_2d = reshape(tensor_3d, [B * H, W]) # 模式4: 转置获取列视图 # [M, N] -> [N, M] transposed = transpose(tensor_2d, 0, 1)

5.2 填充与边界处理

# 使用零填充的切片 padded_view = slice( tensor, shape=[128, 128], offset=[100, 100], valid_shape=[50, 50], # 实际数据区域 pad_value=PadValue.zero # 边界用0填充 ) # 简写形式 padded_view = slice( tensor, shape=[128, 128], offset=[100, 100], pad_value=0 # 简写：0 -> PadValue.zero )

⚠️ 重要提示： manual_dep 参数用于将张量排除在 OverlapMap 自动依赖追踪之外。 这是一个张量生命周期级别的粒度控制， 与作用域级别的 with pl.manual_scope(): 和参数级别的 pl.no_dep(t) 正交。
# 创建手动管理的张量（不参与自动依赖追踪） manual_tensor = create( [1024, 512], DataType.FP32, manual_dep=True ) # 使用显式依赖边提交任务 # pl.submit(task_fn, ..., deps=[manual_tensor])

5.5 性能优化建议

🚀 性能优化最佳实践

  1. 避免不必要的 reshape - reshape 操作在某些后端可能有性能开销
  2. 使用适当的 layout - 根据计算模式选择 ND 或 DN 布局
  3. 批量操作 - 优先使用向量化操作而非逐元素循环
  4. Span 管理 - 在性能关键路径中避免频繁的 span 捕获
  5. 数据类型匹配 - 确保操作数数据类型一致，避免隐式转换

""" 滑动窗口示例：从大张量中提取多个2D窗口 """ # 原始输入张量 [B, H, W, C] input_tensor = create([B, H, W, C], DataType.FP32) # 滑动窗口参数 window_size = 3 stride = 2 # 提取窗口（简化的伪代码） windows = [] for i in range(0, H - window_size + 1, stride): for j in range(0, W - window_size + 1, stride): window = slice( input_tensor, shape=[B, window_size, window_size, C], offset=[0, i, j, 0], drop_dims=[0] # 丢弃batch维度，输出 [window_size, window_size, C] ) windows.append(window)

6.3 分块矩阵运算

""" 分块矩阵乘法：将大矩阵分成小块进行运算 """ # 原始大矩阵 large_matrix = create([M, N], DataType.FP32) # 分块参数 block_size = 64 # 获取块数 num_blocks_m = M // block_size num_blocks_n = N // block_size # 提取单个块 block_ij = slice( large_matrix, shape=[block_size, block_size], offset=[i * block_size, j * block_size] ) # 对块进行操作后写回 processed_block = exp(block_ij) # 逐元素指数 result = assemble(large_matrix, processed_block, offset=[i * block_size, j * block_size])
